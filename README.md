# Jeu python
Groupe On est là

Dans le cadre de ce projet de groupe de notre premiere année de classe préparatoire intégrée nous avons réalisé les essais d'un jeu du style "shoot them up". Nous avons donc utilisé python et l'extansion pygame pour arriver a nos fins, les grahismes ne nous etant pas imposés nous avons déja reussi a faire apparaitre le joueur sur l'écran et a restreindre ses positions sur l'axe horizontal. Nous avons également reussi a lui faire tirer des munitions sans lui donner pour de moment de recharge entre celles-ci. Le projet est loin d'être fini mais nous comptons essayer de le continuer si possible pour effectuer le système de points les ennemis et ajouter des textures pour l'aspect estethique.
Le jeu se lance pour le moment en lançant la simulation sur pygame et une interface graphique serait d'un meilleur effet
Dans ce projet, Clément etait chargé d'effectuer le joueur et ses projectiles, Lucie et Antoine etaient chargés d'effectuer les ennemis et Ismael et Titouan, le système de points et eventuellement les textures.
